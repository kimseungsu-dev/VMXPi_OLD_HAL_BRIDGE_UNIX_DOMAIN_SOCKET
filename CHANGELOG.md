# Changelog

## 1.0.0 - 2026-07-12

Initial stable release.

- ARMHF VMX-pi HAL controller
- Native ARM64 Unix Domain Socket client
- VMX resource and protocol support
- Output safe-stop and watchdog handling
- VMX RTC and Linux clock synchronization
- systemd startup and crash recovery
- CPU affinity configuration
- Automated installer and uninstaller
- Healthcheck, smoke test and final audit
